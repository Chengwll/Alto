🤖 03 Agent 应用层 · AI智能员工
🔗
多Agent 协同作战
面对复杂业务场景，多个专业Agent按流程分工协作——一个Agent完成分析后自动触发下一个Agent执行，形成从洞察→诊断→策略→执行的完整链路
竞品监控→情报分析→策略建议 客户跟进→盘客评估→话术推送
🎯
单Agent 精准服务
面对单一明确需求，一个Agent独立完成全流程——调用自身知识库与工具，从理解意图→检索数据→生成结果一步到位
销售战况一键分析 报表订阅自动生成

🚀
项目初始化 Agent 营销总 · 策划负责人

🧩 五大核心配置
💬
提示词系统
Agent 的"人设"
  • 角色定义：身份、语气、专业领域
  • 行为约束：红线规则与安全边界
  • 输出规范：格式、长度、结构要求
  • 少样本示例：典型问答模板
📚
知识库系统
Agent 的"知识"
  • 项目档案：楼盘信息、户型卖点
  • 业务文档：SOP、话术库、FAQ
  • 竞品资料：市场报告、价格表
  • 自动更新：文档变更后索引重建
⚡
技能系统
Agent 的"能力"
  • 分析类：战况分析、渠道诊断
  • 生成类：报表产出、文案创作
  • 监控类：竞品异动、团队绩效
  • 技能编排：多技能串联与触发
🔧
工具系统
Agent 的"手脚"
  • 数据工具：CRM查询、ERP数据
  • 通信工具：企微/钉钉/飞书推送
  • 检索工具：联网搜索、竞品抓取
  • 权限管控：最小权限+审计日志
🛡️
数据管控系统
Agent 的"围栏"
  • 数据范围：限定可访问的数据域
  • 人员范围：限定可用角色与名单
  • 敏感脱敏：手机/身份证自动脱敏
  • 操作审计：全链路访问日志
🤝 共建共创模块
📋 能力来源 Agent 与 Skill 从何而来
📦系统内置
平台预置的标准化能力，由产品团队持续维护升级——开箱即用，覆盖项目建档、客户跟进、战况分析、报表订阅等高频场景，零配置即可上线
🏗️区域共建
内部员工基于业务经验自建——将区域特色流程、本地化话术、定制化分析沉淀为可复用能力，好经验从个人技巧变为组织资产
🏪 能力市场 浏览、选用与发布能力的统一入口
🤖Agent 市场
AI 智能员工的统一交易中心——每个 Agent 拥有独立的角色人设、知识边界与工具权限，可独立运行也可多Agent协同，按业务场景分类浏览，一键选用
⚡Skill 市场
可组合业务能力模块的交易中心——每个 Skill 是一个完成特定任务的独立能力单元，可被多个 Agent 复用调用，灵活组合构建复杂业务流
任务 → AI 核心
